# Bo OpenSearch

一个**可广泛应用**的搜索 skill。无论你是查学术资料、追实时新闻、找代码答案、对比商品，还是搜操作教程，它都能上。

它做的事很纯粹——在你搜索的时候，主动抓取真实网页内容，按相关性、深度、时效三个维度排序，把无关项、广告软文、内容农场镜像、打不开的死链全部剔除，最后只把有价值的部分按干净格式呈现。多源冲突会明确标出，含推广的结果会打上标签，需要登录或被 paywall 挡住的页面也会如实告知，绝不编造来源。

---

## 用法

### 1. 安装

确保 [skill.md](skill.md) 在你的 skills 目录即可。

### 2. 自动触发

安装后无需任何额外操作。只要你在和 Claude 聊天时出现以下关键词，skill 就会自动触发：

- `搜搜`
- `搜索`
- `查一下`
- `查查`
- `search`

例如直接说："**搜搜** 2026 最值得买的笔记本"，它就会自动接管。

### 3. 主动触发

如果想明确指定用这个 skill，在命令前加 `/bo-opensearch` 即可：

```
/bo-opensearch RAG 的最新研究进展
```

## 适合谁

- **学生 / 研究者** — 直接拿到权威来源，跳过 SEO 农场

- **普通用户** — 商品对比时一眼分清独立评测与软文
